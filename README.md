# Robust Image Segmentation for Foliar Disease Identification

This system detects and classifies tomato leaf diseases via a GUI. It processes images using noise filtering and contrast enhancement, applying dual segmentation to isolate infections. It classifies the disease type, calculates severity, and provides actionable treatments for an automated, end-to-end diagnosis pipeline.

##  Tools & Technologies
* Development Environment: MATLAB
* Core Techniques:Spatial filtering, contrast stretching, image segmentation, morphological operations

##  Key Features
* Dual Segmentation: High-accuracy boundary detection for isolating complex foliar infections.
* Interactive GUI: Users can cleanly upload, process, and analyze leaf images seamlessly.
* Severity Analytics: Calculates the exact surface area percentage of leaf damage.
